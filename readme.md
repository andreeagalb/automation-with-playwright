## linux commands
```bash
cd == change directory
pwd == arata in ce director esti
ls == list toate fisierele/directorele din directorul curent
ls -la == la fel ca ls doar ca iti da mai multe detalii
code pom.xml == Opens just the pom.xml file in your current VS Code window.
code . == Opens the entire current folder as a project in VS Code
code -r pom.xml	== Opens the file and reuses the active window
code App.java	== Opens your Java source code file.
```
